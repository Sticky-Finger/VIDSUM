> 先解释下`src/lib/utils.ts` - cn 工具函数是什么？ 

⏺ cn 工具函数是 React + Tailwind CSS 项目中非常常用的一个工具，用于合并 CSS
   类名。

  作用

  它的主要作用是智能地合并多个类名，同时处理 Tailwind CSS 的样式冲突。

  使用场景

  例如，你有一个按钮组件，它有默认样式，但有时需要覆盖某些样式：

  // 没有 cn 函数时
  <button className="px-4 py-2 bg-blue-500 text-white disabled:bg-gray-300 
  disabled:cursor-not-allowed">
    按钮
  </button>

  // 有 cn 函数时
  <button className={cn(
    "px-4 py-2 bg-blue-500 text-white",
    disabled && "disabled:bg-gray-300 disabled:cursor-not-allowed"
  )}>
    按钮
  </button>

  为什么需要它？

  1. 条件类名：可以根据 props 动态添加/移除类名
  2. Tailwind 冲突处理：twMerge 能智能处理 Tailwind 类的冲突

  // 例如：默认有 p-4，但想覆盖为 p-2
  cn("p-4 bg-white", "p-2")
  // twMerge 会输出："p-2 bg-white" 而不是 "p-4 bg-white p-2"

  依赖

  - clsx - 处理条件逻辑，可以传入 true/false/undefined 等
  - tailwind-merge - 智能合并 Tailwind 类，解决冲突

  这在 shadcn/ui 组件库中是标准配置，几乎是现代 React + Tailwind
  项目的标配工具。

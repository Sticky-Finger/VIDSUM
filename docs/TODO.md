## 计划中 📋

### P0 功能（必须有）

- [ ] **输入与任务创建**
    - [ ] 选择本地音视频文件（支持 mp4, mkv, mov, mp3, wav, m4a 等）
    - [ ] 选择本地字幕文件（.srt / .vtt）作为起点

- [ ] **音频转文字 (ASR)**
    - [ ] 本地 Whisper 引擎，支持 Tiny、Base 两种规格模型切换
    - [ ] 云端 ASR 预留 OpenAI 兼容接口配置（API URL + Key + Model）
    - [ ] 转写过程显示实时进度条

- [ ] **字幕确认与预览**
    - [ ] 转写完成后展示字幕预览区域（时间轴 + 文本）
    - [ ] 用户手动点击"确认并进入总结"，将当前字幕作为总结输入源

- [ ] **大模型总结**
    - [ ] 用户配置 OpenAI 兼容 API（Base URL、API Key、Model ID）
    - [ ] 点击"生成总结"后，系统使用内置固定 Prompt 提交字幕全文
    - [ ] 输出包含 `[HH:MM:SS]` 时间戳锚点的分层 Markdown

- [ ] **结果展示与导出**
    - [ ] Markdown 富文本预览（语法高亮、层级缩进）
    - [ ] 支持导出为 `.md` 文件保存至本地

## 暂停 ⏸️

### P2 功能（未来考虑）

- [ ] **官方字幕自动提取**：不主动探测内嵌字幕轨，不自动下载网络字幕
- [ ] **说话人分割**：转写字幕中不包含说话人标签
- [ ] **视频预览播放**：应用内不嵌入视频播放器
- [ ] **自定义 Prompt 编辑**：使用固定内置 Prompt
- [ ] **批量处理与历史记录**：不支持多文件队列处理，不保存历史任务记录
- [ ] **字幕内容编辑**：字幕预览区为只读模式
